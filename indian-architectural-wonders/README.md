# Indian Architectural Wonders

A visual journey through India's magnificent architectural heritage spanning from the 1st century to the 21st century.

## About

This project showcases 25 iconic monuments across India through an interactive timeline and map interface. All photographs are captured during travels by Madhu and Shashi ([@madfortravels](https://instagram.com/madfortravels)).

## Features

- **Interactive Timeline**: Chronological view of monuments from 1st to 21st century
- **Interactive Map**: Real OpenStreetMap showing exact locations of all monuments
- **Image Galleries**: Multiple photographs for each monument
- **Monument Details**: Historical information and Wikipedia links
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Monuments Covered

The collection includes:
- Sanchi Stupa (1st Century)
- Ajanta Caves (2nd-3rd Century)
- Aihole, Badami Caves, Pattadakal (4th-7th Century)
- Ellora Caves, Khajuraho Temples (8th-10th Century)
- Belur, Halebidu, Rameshwaram Temple (11th-13th Century)
- Qutub Minar, Gol Gumbaz (13th-17th Century)
- Golconda Fort, Agra Fort (16th Century)
- Hampi (16th Century)
- Dwarakadish Temple, Golden Temple (18th-19th Century)
- Taj Mahal, Red Fort (17th Century)
- Mysore Palace (19th Century)
- Dakshineshwar Temple (19th Century)
- India Gate (20th Century)
- Statue of Unity (21st Century)

## Technologies Used

- HTML5, CSS3, JavaScript
- Leaflet.js for interactive maps
- OpenStreetMap for map tiles

## Running Locally

1. Clone this repository
2. Open `index.html` in a web browser, or
3. Run a local server:
   ```bash
   python -m http.server 8080
   # or
   npx http-server -p 8080
   ```
4. Visit `http://localhost:8080`
